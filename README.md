# AI Heart Risk Project

This project predicts **heart attack risk** from **retinal fundus images** using **deep learning**.

- **Author:** Deepa Lakshmi
- **Files:** Retainal.ipynb, model.keras
- **Tech Stack:** Python, Keras, TensorFlow, OpenCV, Matplotlib
- **Dataset:** Retinal fundus images
- **Purpose:** Early detection of heart disease from eye scans
